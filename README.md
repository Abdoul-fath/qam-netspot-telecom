# Projet de Modulation Numérique

Ce dépôt contient deux projets réalisés dans le cadre du module de Modulation Numérique :

- 🎛️ **Projet 1 : Modulation QAM**
- 📶 **Projet 2 : Étude de site Wi-Fi avec NetSpot**

---

## 🎛️ Projet 1 : Modulation QAM

### Description

Le projet QAM simule la modulation et la démodulation de signaux numériques avec différentes valeurs de M (4-QAM, 16-QAM, etc.). Deux volets ont été réalisés :
- **QAM binaire** : génération aléatoire de bits, modulation, ajout de bruit, et calcul du BER.
- **QAM image** : modulation et transmission d'une image avec affichage de l'image modulée et reçue.

Une interface web (Flask) a été développée pour visualiser les résultats de manière interactive.

### Contenu

- `qam_modulation.py` : Modulation de bits.
- `qam_image_modulation.py` : Modulation d'image.
- `app/` : Application Flask pour visualiser les modulations.

## 📶 Projet 2 : Étude de site Wi-Fi (NetSpot)

### Description

Le projet NetSpot consiste à :
- Effectuer une **enquête de site Wi-Fi** dans une zone réelle.
- Réaliser une **planification** de couverture optimale avec les bons canaux et puissances.

### Contenu

- `netspot_enquete.pdf` : Analyse du réseau existant (mesures, heatmaps).
- `netspot_planification.pdf` : Proposition de déploiement optimisé.
- 📹 `video_netspot_explication.mp4` : Présentation en vidéo du projet.

---

## 🛠️ Technologies utilisées

- Python (NumPy, Matplotlib, Flask)
- HTML/CSS pour l'interface
- NetSpot (outil de survey Wi-Fi)
- VS Code pour le développement
